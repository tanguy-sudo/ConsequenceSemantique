## Installation
***
Pour installer le projet vous pouvez le cloner avec une clé SSH, ou bien télécharger le dossier ZIP.
```
$ git clone git@github.com:tanguy-sudo/ConsequenceSemantique.git

```
**Pour compiler**
```
$ gcc examen.c  -Wall -O3 -o examen.exe
```
